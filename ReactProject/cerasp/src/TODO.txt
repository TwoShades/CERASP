CSS:
Color info in one stylesheet
Imported in global App.js
All components will require a color class when they are instantiated
(need naming convention)
ex: bgColor-textColor-theme
ex: blue-white-theme, white-beige-theme, etc.

Define/adjust colors in one file, so much easier further down the road.

ex:

==========BASIC COMPONENT==========
const BannerPanel = ({ theme }) => {
return <div className={`banner-panel ${theme}`}>Banner Content</div>;
};

==========CALL COMPONENT IN APP==========
import "./colors.css";
import ScrollPage from "./ScrollPage";
import SideBySidePanel from "./SideBySidePanel";
import BannerPanel from "./BannerPanel";

function App() {
return (
<ScrollPage>
<SideBySidePanel
left={<BannerPanel theme="white-beige-theme" />}
right={<BannerPanel theme="blue-white-theme" />}
/>
</ScrollPage>
);
}

export default App;

==========GLOBAL CSS TO DEFINE COLOR THEMES==========
:root {
--blue: #3498db;
--white: #ffffff;
--beige: #f5f5dc;
--red: #e74c3c;
--black: #222;
}

/_ Theme Classes _/
.white-beige-theme {
background-color: var(--white);
color: var(--beige);
}
s.blue-white-theme {
background-color: var(--blue);
color: var(--white);
}

.red-black-theme {
background-color: var(--red);
color: var(--black);
}

:root is where we can define the color swatch, call those in different themes, pass those into components (background-color and color)

Components can further specify things like border color or whatever where needed
